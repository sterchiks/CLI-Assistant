# 🤖 CLI Assistant

**Интерактивный AI-ассистент для терминала**

Пишите естественным языком, а ИИ выполнит за вас команды, управляет файлами и системой. Для тех кто не помнит все консольные команды Linux.

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Textual](https://img.shields.io/badge/textual-0.47.0-green.svg)](https://textual.textualize.io/)
[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/sterchiks/CLI-Assistant/blob/main/LICENSE)

## ✨ Возможности

✅ **Натуральный язык** — пишите как обычно, нет необходимости запоминать команды  
✅ **60+ системных инструментов** — работа с файлами, дисками, процессами, терминалом  
✅ **Безопасность** — все опасные действия требуют подтверждения  
✅ **Встроенный TUI** — красивый интерфейс прямо в терминале  
✅ **Поддержка всех популярных AI** — Claude, Gemini, OpenAI, Groq, Ollama, LM Studio, и все OpenAI-совместимые  
✅ **5 тем оформления** — Dark, Light, Cyberpunk, Nord, Solarized  
✅ **Локальные LLM** — работает с локальными моделями через Ollama  
✅ **Мастер настройки** — простая установка за 1 минуту

## 📦 Установка

### Linux / macOS
```bash
git clone https://github.com/sterchiks/CLI-Assistant.git
cd CLI-Assistant
chmod +x install.sh
./install.sh
```

### Windows
```cmd
git clone https://github.com/sterchiks/CLI-Assistant.git
cd CLI-Assistant
install.bat
```

### Быстрый запуск (без установки)
```bash
./run.sh
```

## 🚀 Использование

Запустите ассистент:
```bash
cli-assistant
```

Или короткий алиас:
```bash
ai
```

### Примеры запросов:

```
> удали все .log файлы в /var/log старше 7 дней
> покажи что занимает больше всего места на диске
> найди все файлы с паролями в домашней директории
> установи nginx и настрои конфиг на порт 8080
> создай резервную копию папки /etc
> перезапусти службу docker
> покажи процессы которые потребляют больше всего памяти
> напиши скрипт бэкапа базы данных PostgreSQL
```

## 🔥 Slash-команды

| Команда                | Описание                                  |
|------------------------|-------------------------------------------|
| `/help`                | Справка по всем командам                  |
| `/settings`            | Открыть полный экран настроек             |
| `/provider <name>`     | Сменить AI провайдер                      |
| `/model <name>`        | Сменить модель                            |
| `/theme <name>`        | Сменить тему оформления                   |
| `/apikey`              | Изменить API ключ                         |
| `/clear`               | Очистить историю чата                     |
| `/cd <path>`           | Сменить рабочую директорию                |
| `/ls`                  | Содержимое текущей директории             |
| `/sudo`                | Запросить пароль sudo на сессию           |

## 🔧 Конфигурация

Конфиг хранится в `~/.cli-assistant/config.json`:

```json
{
  "ai": {
    "provider": "anthropic",
    "model": "claude-3-5-sonnet-20240620",
    "api_key": "sk-ant-...",
    "temperature": 0.7,
    "stream": true
  },
  "ui": {
    "theme": "dark",
    "show_tool_calls": true
  },
  "safety": {
    "confirm_destructive": true,
    "confirm_sudo": true
  }
}
```

Запустить мастер настройки повторно:
```bash
cli-assistant --setup
```

## 🔒 Безопасность

✅ API ключи хранятся в системном keychain, не в plaintext  
✅ Пароль sudo хранится только в памяти, кэш 15 минут  
✅ Все опасные операции (удаление, sudo) требуют явного подтверждения  
✅ Защита от `rm -rf /` и других опасных команд  
✅ Блокировка доступа к системным файлам: `/etc/shadow`, `/etc/passwd`, `/dev/`  
✅ Перед редактированием важных файлов автоматически создаются бэкапы
